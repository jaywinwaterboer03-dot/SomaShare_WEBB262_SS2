## 🚨 Common Issues & Fixes

| Problem | Solution |
|---------|----------|
| Git Changes panel won't open | Restart Visual Studio |
| Stage All button disabled | Ensure files are in project |
| Commit button disabled | Stage files + add message |
| Push fails | Sign in to GitHub first |
| Can't see files | Press F5 to refresh |
